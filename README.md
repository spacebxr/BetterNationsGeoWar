# GeoWarPlugin

GeoWarPlugin is a Paper Minecraft plugin designed for version 1.21.1. It provides a management interface for nation leaders to control their towns, military, economy, and diplomacy, integrating closely with Towny.

It fully supports Bedrock Edition players via Geyser and Floodgate, providing native Bedrock forms for a seamless user experience, while Java players receive traditional inventory-based graphical user interfaces.

## Features

- Nation and town management integration with Towny.
- Diplomatic tools for treaties and declaring war.
- Economic controls to pay citizens and manage taxes.
- Military command interface.
- Native UI support for Bedrock clients (Geyser Forms) and Java clients (Chest GUIs).

## Requirements

- Paper 1.21.1
- Towny
- Geyser-Spigot
- Floodgate
