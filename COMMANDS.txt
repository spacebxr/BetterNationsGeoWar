GeoWarPlugin Commands
======================

Plugin metadata
---------------
Plugin name: GeoWarPlugin
Version: 1.0.0
Author: GeoWar Development Team

General command
---------------
/nationgui
/ngui
    Opens the nation management GUI.

Capture-zone commands
---------------------
Permission: geowar.capturezone.admin

/capturezone create <id> <type> [radius] [capture-seconds]
    Creates a capture zone at the sender's current location.
    Types: OIL_RIG, KOTH, TERRITORY
    Default radius: 40 blocks
    Default capture time: 600 seconds

/capturezone delete <id>
    Deletes a registered capture zone.

/capturezone info <id>
    Shows the zone name, owner, and capture progress.

/capturezone reload
    Reloads capture configuration messaging.

Capture permission
------------------
Permission: geowar.capturezone.capture
Default: true

Players with this permission can contribute to capture progress.
Players must also belong to a Towny town.
